This folders contains test files.

These regular expressions are used to catch errors related with doubles and hexa numbers...

"."{INT}{EXP}{INT}			return ERROR_D; // Not valid exponential...
{DBL}{EXP}[0-9]*"."[0-9]*	return ERROR_D; // Exponential with decimal...
{INT}{EXP}({INT}|{INT}"."|"."{INT})				return ERROR_D; // Exponential with decimal...
({DBL}|{INT}){EXP}|{EXP}	return ERROR_D; // Only E+-...
{DBL}{EXP}[^0-9]+			return ERROR_D;
("0x"|"0X")[0-9a-zA-Z]+		return ERROR_H; // Not valid hexa number...
("0x"|"0X")					return ERROR_H; // Not valid hexa number...
"."{INT}					return ERROR_D; // Not valid double number...
